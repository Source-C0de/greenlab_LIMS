Build a modern, production-ready SaaS frontend for a cloud-based Laboratory Information Management System (LIMS) called "GreenLIMS KSA".

Tech Stack:
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Zustand or React Query for state management
- Fully responsive (desktop + tablet + mobile)

Design Requirements:
- Clean, minimal, enterprise SaaS UI (inspired by Stripe, Vercel, Linear)
- Support Arabic (RTL) and English (LTR) with language switch
- Dark + Light mode
- Use consistent spacing, cards, tables, and modular layouts
- Use charts (Recharts) for dashboards
- Use realistic mock data everywhere

Architecture:
- Multi-tenant SaaS (each lab has its own workspace)
- Sidebar navigation + top header
- Role-based UI rendering (Admin, Lab Manager, Analyst, Client, Accountant)
